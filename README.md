# 피카 피카 !! 피카츄 배구 
### 링글 voice recognizer 스터디 결과물
       
## 개요 

          gorisanson님이 역엔지니어링으로 js로 구현한 피카츄 배구에 speech
          recognition api를 적용해보았습니다. 엔터 대신 피카를 외쳐야 합니다.
          (여러번 플래이를 거친 후 팁을 드리자면 상당히 긴 딜레이를 예상하고
          피카를 외쳐야 합니다 )
